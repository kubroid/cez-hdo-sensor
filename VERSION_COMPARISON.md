# 🔄 Выбор версии CEZ HDO Sensor

В репозитории доступны **2 версии** интеграции:

## 🌟 Ветка `main` - HACS версия
**Рекомендуется для продвинутых пользователей**

### ✅ Преимущества:
- 🔄 **Автообновления** через HACS
- 📦 **Легкая установка** в 1 клик
- 🔧 **Управление версиями** 
- 📊 **Статистика установок**

### ❌ Недостатки:
- 🛠️ Требует **установленный HACS**
- 🔗 Зависимость от **внешнего сервиса**
- 📋 **Сложная отладка** проблем с HACS

### 📦 Установка:
```
HACS → Integrations → Custom repositories → 
https://github.com/kubroid/cez-hdo-sensor
```

---

## 🏠 Ветка `builtin-integration` - Встроенная версия  
**Рекомендуется для всех пользователей**

### ✅ Преимущества:
- 🚀 **Супер простая установка** - скопировать папку
- 🛡️ **Стабильная работа** - без зависимостей
- ⚡ **Быстрый запуск** - работает сразу
- 🔧 **Легкая отладка** - все файлы локально

### ❌ Недостатки:
- 🔄 **Ручные обновления** - нужно скачивать новые версии
- 📦 **Ручная установка** - копирование файлов

### 📦 Установка:
```bash
wget https://github.com/kubroid/cez-hdo-sensor/archive/builtin-integration.zip
# Распаковать в /config/custom_components/
```

---

## 🎯 Какую выбрать?

### Для новичков: 🏠 **Встроенная версия**
- Простая установка
- Стабильная работа
- Не нужно изучать HACS

### Для опытных: 🌟 **HACS версия**  
- Автообновления
- Интеграция с HACS экосистемой
- Удобное управление

| Параметр | HACS версия | Встроенная версия |
|----------|-------------|-------------------|
| **Версия** | 1.1.0 | 2.0.0 |

| **Функционал** | Одинаковый | Одинаковый |
| **Код интеграции** | Идентичный | Идентичный |

## 🚀 Переключение между версиями

### Из HACS в встроенную:
1. Удалите через HACS
2. Скачайте встроенную версию
3. Скопируйте в custom_components

### Из встроенной в HACS:
1. Удалите папку cez_hdo
2. Установите через HACS
3. Перенастройте интеграцию

---

**Рекомендация**: Начните с **встроенной версии** (`builtin-integration`) - она проще и надежнее!
